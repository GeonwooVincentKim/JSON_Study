# webserver-express
Simple Webserver with Express

# 1. clone

```
$ cd ~/Desktop
$ git clone https://github.com/ungmo2/webserver-express.git
```

# 2. install express 

```
$ npm i
```

# 3. start webserver

```
npm start
```

# 4. browser

```
localhost:3000
```

# 5. root directory

webserver-express/public
